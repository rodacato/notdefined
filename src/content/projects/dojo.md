---
name: Dojo
tagline: El dojo para devs que todavía tienen algo que probarse. A sí mismos.
product: >-
  Práctica diaria contra el vibe coding. Entras, te tocan 3 kata —sin skip, sin
  reroll—, eliges uno y lo trabajas sin AI mientras corre el timer. Un sensei
  (un LLM con un rol y experiencia específicos) te evalúa en tiempo real con
  feedback honesto, no con la respuesta ni con aplausos. No es certificación ni
  leaderboard: es mantenerte técnicamente vivo.
technical: >-
  Monorepo pnpm con React 19 + Vite en el front y una API en Hono sobre Node,
  PostgreSQL de datastore. Evaluación en tiempo real contra un LLM con personas
  por dominio. Calidad medida con SonarQube por workspace y CI en GitHub Actions.
stack:
  - TypeScript
  - React
  - Hono
  - Node.js
  - PostgreSQL
  - Vite
  - pnpm monorepo
  - LLM
