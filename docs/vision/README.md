# Vision — notdefined.dev

> The north star and hard rules. If a proposal conflicts with this document, the proposal loses.

---

## North star

A personal technical blog where Adrian writes what he wants to remember, in his own voice, with no compromise on the writing being his.

The name `notdefined` is the thesis. `not defined` is what happens when something exists but hasn't been given a value yet — ideas in progress, things about to become something. The trailing blinking `_` cursor in the wordmark is the visual proof.

---

## Hard rules

1. **Every blog post passes through [`audience-panel.md`](../editorial/audience-panel.md) before `draft: false`.** No exceptions. The panel exists to break the post before a real reader does.
2. **Editorial content is español mexicano casual; repo artifacts are English.** Three layers — see [`.kwik-e/memory/feedback_repo_language.md`](../../.kwik-e/memory/feedback_repo_language.md). The blog reads in Spanish street; the codebase reads in clean English; the chat between Adrian and the AI is private and Spanish.
3. **No co-author attribution.** Adrian is the sole author of every post, every commit, every artifact. No "Generated with Claude", no co-author trailers, no AI footnotes.
4. **No comments section, no newsletter, no signup form.** Adding any of those needs an explicit ADR — not "porque sería cool".
